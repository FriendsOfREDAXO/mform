# ImageList-Widget

Die ImageList ist ein Widget, das es erm√∂glicht, eine Liste von Bildern zu erstellen.

## Modul-Eingabe

```php
<?php
// init mform
echo MForm::factory()
    // add fieldset area
    ->addFieldsetArea('MForm Widgets', MForm::factory()
        // custom elements
        ->addImagelistField(1, ['label' => 'Image List'])
    )
    // parse form
    ->show();
```
